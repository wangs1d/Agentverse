# Agent World 商业智能推广 Demo - 产品需求文档（PRD）

## 1. 产品概述

Agent World 商业智能推广 Demo 是 Agent World 平台在「私人 Agent 智能体」应用基础上新增的 **B 端商业智能推广模块**，用于演示「B 端商家 → 算法匹配中台 → C 端私人 Agent」三方闭环推广链路。Demo 旨在以最具科技感与极简美学的可视化方式，展示 AI Agent 理解用户画像、精准匹配产品并以自然对话口吻进行个性化推荐的核心能力，是平台面向 B 端商家招商与商业化探索的核心 Demo。

- **目标用户**：Agent World 平台运营方、B 端潜在合作品牌方、内部产品/技术决策者
- **核心价值**：以一站式可视化方式呈现 Agent 个性化推荐的差异化能力，加速商业合作签约

## 2. 核心功能

### 2.1 用户角色
| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 访客（演示用户） | 平台导航栏点击「商业智能推广」入口 | 浏览并完整体验推广闭环演示 |

### 2.2 功能模块
1. **入口按钮模块**：在已有 Agent World 应用导航栏中嵌入「商业智能推广」入口按钮，支持快速进入演示界面。
2. **三栏式演示主界面**：
   - **左栏 - B 端商家上传区**：产品类型 Tab 切换、产品信息表单（名称/描述/图片/价格）、提交推广按钮。**不直接选画像**，由算法自动推断。
   - **中栏 - 算法匹配中台（两阶段）**：
     - 第一阶段：**AI 产品画像推断** —— 基于产品信息自动生成目标用户画像标签（含"AI 推断中"动效 + 标签逐个出现）
     - 第二阶段：**双向匹配可视化** —— 产品画像 ↔ 平台用户池双向连线动画、匹配度进度条、数据流动效、匹配数汇总
   - **右栏 - C 端私人 Agent 推荐区**：模拟私人 Agent 聊天界面，匹配成功后 Agent 以自然口吻向用户推荐产品。
3. **可独立控制模块**：每个分栏支持收缩/展开，演示者可聚焦查看任意模块。

### 2.3 页面细节
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 入口 | 商业智能推广按钮 | 嵌入导航栏，hover 微交互，点击进入演示页 |
| 演示主页 | 左栏 B 端商家上传区 | 切换软件/硬件 Tab；填写产品信息（名称、描述、图片、价格）；提交后触发算法分析 |
| 演示主页 | 中栏算法匹配中台 | 阶段一：AI 推断产品目标用户画像（带算法加载 + 标签涌现动效）；阶段二：画像 ↔ 用户池双向连线 + 匹配度进度条 + 匹配数统计 |
| 演示主页 | 右栏 C 端 Agent 推荐区 | 模拟聊天窗口：Agent 根据用户画像生成自然推荐话术，支持逐条打字机动画 |

## 3. 核心流程

```mermaid
flowchart LR
    A[用户点击入口按钮] --> B[进入三栏演示界面]
    B --> C[B端填写产品信息 不选画像]
    C --> D[点击"提交推广"]
    D --> E[左栏高亮 + 提交确认]
    E --> F1[中栏阶段一: AI产品画像推断]
    F1 --> F2[算法涌现目标用户标签]
    F2 --> G[中栏阶段二: 产品画像与用户池双向匹配]
    G --> H[匹配成功: 显示匹配Agent数]
    H --> I[右栏激活: Agent开始对话推荐]
    I --> J[Agent以自然口吻输出推荐]
    J --> K[用户可重置并重新演示]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深灰 `#1a1a1a` 为底色，搭配近黑 `#0d0d0d` 与中灰 `#2a2a2a` 营造层次
- **点缀色**：科技蓝 `#0066cc`，高亮态使用浅蓝 `#3399ff` 与霓虹蓝 `#00aaff`
- **辅助色**：成功绿 `#10b981`、警告橙 `#f59e0b`、纯白文字 `#f5f5f7`、次级灰 `#8e8e93`
- **按钮风格**：低圆角（4–8px）矩形按钮，主按钮科技蓝实心 + 微弱辉光；次按钮描边 + hover 浅高亮
- **字体**：
  - 标题：SF Pro Display / Inter Display（系统优先），字重 600/700，字号 14/16/20/28
  - 正文：SF Pro Text / Inter，字重 400/500，字号 12/14
  - 等宽（数据/代码）：JetBrains Mono
- **布局风格**：顶部极简导航 + 三栏可独立折叠的卡片化布局；卡片使用 1px 边框 + 弱阴影，无圆角过度的"卡片感"
- **图标/表情风格**：使用 lucide-react 线性图标，统一 1.5px 描边，禁用 emoji
- **空间感**：信息层级清晰，留白克制；用细线分割层级而非色块

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|--------|
| 入口 | 商业智能推广按钮 | 蓝色线性图标 + 文字，hover 时图标轻微旋转 + 文字变色 |
| 演示主页 | 顶部状态栏 | 三段式步骤指示器：上传 → 匹配 → 推荐 |
| 演示主页 | 左栏 | 顶部 Tab 切换（软件/硬件） + 玻璃感表单 + 标签云 + 主操作按钮 |
| 演示主页 | 中栏 | 中央圆形算法节点 + 周围标签散点 + 双向动画连线 + 顶部数据指标 + 底部匹配度进度条 |
| 演示主页 | 右栏 | 模拟聊天窗口：用户消息（右侧浅蓝），Agent 消息（左侧深灰卡片 + 头像）|

### 4.3 响应式
- 桌面优先（1280px+），三栏等宽布局
- 中等屏幕（1024–1280px）支持两栏 + 折叠
- 不针对移动端做重度优化（桌面端演示场景）

### 4.4 动效设计
- **页面加载**：标题淡入 + 卡片逐个上滑入场（stagger 80ms）
- **匹配过程**：连线粒子流从左栏标签 → 中枢 → 命中画像的节点，循环动画
- **匹配成功**：中枢节点脉冲放大，匹配数从 0 → 最终值（数字滚动动画）
- **推荐对话**：Agent 消息逐字打字机输出，输出时呼吸光点闪烁
- **按钮 hover**：背景色 200ms 渐变 + 微弱辉光
- **折叠/展开**：宽度 300ms ease 过渡 + 内容淡入淡出
